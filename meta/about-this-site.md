## **About This Site**

This documentation is both a working notebook and a public operations reference. The primary audience is still me as an operator, but the site is organized so other people can follow the platform baseline without having to sift through every experiment and side note.

---

### **Mixed notebook, server-first**

The site intentionally mixes two things:

- A server-operations handbook for the current Proxmox, Debian, Docker, networking, security, and observability stack
- A smaller set of personal reference pages, experiments, and supporting notes that are useful to keep in the repository but are not central to day-to-day platform work

The navigation now prioritizes the first category. Peripheral content can still exist in the repo without competing for first-click attention.

---

### **What lives outside the main flow**

Some material remains intentionally de-emphasized:

- personal notebook content such as games or blog experiments
- scaffolding pages kept for future authoring
- hidden meta pages such as the [Changelog](changelog.md)

That content is still useful context, but it should not dilute the server-documentation path on the home page or in the primary navigation.
